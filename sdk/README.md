## Quick Setup

### Install

Install these dependencies over:

npm:

```shell
npm install @devoltxyz/devolt-protocol
```

yarn:

```shell
yarn add @devoltxyz/devolt-protocol
```

### Build From Source

1. Clone the project:

```shell
git clone https://github.com/devoltxyz/devolt-protocol.git
```

2. Install dependencies:

```shell
cd devolt-protocol
npm install
```
